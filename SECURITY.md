# Security Policy

## Reporting a Vulnerability

Please **do not** report security vulnerabilities through public GitHub issues.

Instead, open a [GitHub Security Advisory](https://github.com/zubairmetabox/animascroll/security/advisories/new) so it can be reviewed privately before a fix is released.

Include as much detail as possible:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge receipt within 48 hours and aim to release a fix within 14 days for confirmed issues.
